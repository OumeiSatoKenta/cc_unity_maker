# ゲームアイデアソース

出典: https://github.com/OumeiSatoKenta/cc_game_ideas

## プロジェクト概要

- ゲームアイデア100本リストをUnityで実装するワークフロー
- cc_game_ideasリポジトリのIdeasをGitHub Issuesに登録
- 順次Unityで実装していく

## ワークフローの目的

1. `cc_game_ideas/ideas/ideas.md` のアイデアをUnityで実装できる仕様まで膨らませる
2. 実装仕様をGitHub Issuesとして登録する
3. Issueをベースに順次Unityプロジェクトを実装する

## ゲームアイデア概要

合計100本のゲームアイデア（6カテゴリ）:

- パズル系 (001-020): 20本
- アクション/アーケード系 (021-040): 20本
- カジュアル/ハイパーカジュアル系 (041-060): 20本
- 放置/クリッカー系 (061-070): 10本
- リズム/音楽系 (071-080): 10本
- 育成/シミュレーション系 (081-090): 10本
- ユニーク/実験的 (091-100): 10本

## 工数分布

- S (1日程度): 多数
- M (1週間程度): 多数
- L (2週間程度): 少数

- アイデア管理: https://github.com/OumeiSatoKenta/cc_game_ideas
