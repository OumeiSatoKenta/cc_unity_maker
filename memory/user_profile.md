## ロール
社内でゲーム量産ワークフローを設計・構築するオーナー。

## 背景
- 対象ユーザーは社内の非エンジニア（プログラミング経験なし）
- 非エンジニアが Claude Code を使って Unity ゲームを自力で完成させる仕組みを作りたい
- Unity プロジェクトは GitHub で管理
- ステータス管理は GitHub Projects（スプレッドシート感覚）を選定

## 設計上の重要な決定
- Unity プロジェクトは毎回作らず、1つのプロジェクトにシーンを追加していく方式
- TopMenu はカテゴリ別タブのミニゲーム集スタイル
