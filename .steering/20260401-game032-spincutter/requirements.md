# Requirements: Game032_SpinCutter

## ゲーム概要

回転する刃の軌道（半径・速度）をスライダーでコントロールし、配置された敵を効率よく一掃するパズルアクション。

## コアループ

1. スライダーで刃の回転半径と速度を調整する
2. 発射ボタンで刃を放つ
3. 刃が敵に接触すると敵を撃破
4. 全敵撃破でクリア
5. 発射回数を使い切っても残敵がいるとゲームオーバー

## 画面構成

- ゲーム画面（敵配置・刃・軌道プレビュー・スライダー）
- クリア画面（★評価）
- ゲームオーバー画面

## 必要な GameObject 一覧

| 名前 | 役割 | コンポーネント | スクリプト |
|------|------|----------------|-----------|
| Blade | 回転する刃 | SpriteRenderer, CircleCollider2D(Trigger) | BladeController |
| Enemy_0〜N | 敵 | SpriteRenderer, CircleCollider2D | Enemy |
| Pivot | 刃の回転中心（非表示） | Transform | - |
| OrbitPreview | 軌道プレビュー（破線円） | LineRenderer | - |
| GameManager | ゲーム状態管理 | - | SpinCutterGameManager |
| SpinCutterManager | コアメカニクス | - | SpinCutterManager |
| Canvas | UI | Canvas, CanvasScaler | - |
| SpinCutterUI | UI管理 | - | SpinCutterUI |

## クリア / ゲームオーバー条件

**クリア条件**: 全敵撃破（残敵数 == 0）
**ゲームオーバー条件**: 発射回数（3回）を使い切っても残敵がいる場合

## 操作仕様

- 半径スライダー (Slider): 0.5 〜 4.0 の範囲で刃の回転半径を調整
- 速度スライダー (Slider): 0.3 〜 2.0 の範囲で刃の角速度（rad/s）を調整
  ※速度は実際には 1.0 〜 6.0 rad/s (π/s ≈ 180°/s 〜 360°/s)
- 発射ボタン: 刃を発射（回転開始）
- 発射中はスライダー操作不可

## ステージ設計（固定配置）

敵8体を画面上に配置（軌道半径を変えれば届くよう設計）:

```
敵位置（ワールド座標, Pivot=(0, 0.5)基準）:
  (-2.0,  3.5), (0.0, 4.0), (2.0, 3.5)   # 上段
  (-3.5,  1.5), (3.5, 1.5)               # 中段
  (-2.5, -1.0), (0.0, -0.5), (2.5, -1.0) # 下段
```

Pivot から各敵への距離範囲: 1.5 〜 3.7 → スライダー 0.5〜4.0 で全滅可能

## ★評価基準

- ★★★: 1回の発射で全滅
- ★★ : 2回の発射で全滅
- ★  : 3回の発射で全滅
