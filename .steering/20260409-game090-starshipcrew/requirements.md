# Requirements: Game090v2_StarshipCrew

## ゲーム概要
宇宙船のクルーを育成し、銀河探検ミッションをこなす育成シミュレーション。クルーの相性やスキルの組み合わせで結果が変わる。最適な編成を見つけて全ミッション制覇を目指す。

## コアループ
1. ミッションにクルーを選んで派遣ボタンを押す
2. ミッション結果（成功/失敗/完璧）が表示され、スコアと経験値を獲得
3. クルーが成長（スキル上昇）→次のミッションに備える
4. 全ミッションを12以上クリアでステージクリア

## クリア条件・ゲームオーバー条件
- **ステージクリア**: そのステージの必要ミッション数以上をクリアする
- **最終クリア**: 5ステージ全てクリア（各ステージ3〜7ミッション）
- **ゲームオーバー**: 連続ミッション失敗が規定数（ステージによる）に達した場合

## ゲーム深化設計

### 判断ポイント
- **誰を選ぶか**: ミッションの必要スキル（戦闘/エンジニア/医療）に合わせてクルーを選ぶか、相性シナジーを優先するか
- **何人で行くか**: 1〜3人を選べる。多いほど成功率は上がるが次のミッションに備えられるクルーが減る
- **ミッション順序**: 簡単なミッション（成功率高）から先にこなして経験値を稼ぐか、高難度（高報酬）を先に狙うか
- **回復 vs 即派遣**: 被ダメクルーを休ませるか（スキル低下中）そのまま次のミッションへ送るか

### スキルカーブ
- うまいプレイヤー: 相性シナジーを事前に把握し、3人コンボで×1.5ボーナスを安定して発動
- 下手なプレイヤー: スキル値だけで選んで相性ペナルティが発生し、成功率が下がりミッション失敗が増える

### ゲームデザインパターン
**リソース管理 + 連鎖/波及効果**を採用:
- クルーの「体力」「スキル」「連続派遣可能回数」という3つのリソースを管理
- 相性システムが連鎖：ペアAとBが良相性 → さらにBとCが良相性 → 3人コンボで×1.5の波及ボーナス

### コンボ/乗算スコアシステム
- 連続ミッション成功でコンボカウント増加
- combo=1: ×1.0、combo=2: ×1.2、combo=3: ×1.5、combo=4+: ×2.0
- 完璧クリア（成功率90%以上のミッションを選択）で追加+50pt
- 相性シナジー2人: +30pt、3人コンボ: +80pt

### ステージごとの新ルール追加
- Stage 1: 基本派遣のみ。成功/失敗の結果のみ
- Stage 2: 【相性システム】クルー間の相性が成功率に影響（良相性+20%、悪相性-15%）
- Stage 3: 【装備システム】ミッション前に装備を1つ選択可能。スキル補正を付与
- Stage 4: 【緊急イベント】ミッション中にランダムでイベント発生（隕石回避など）→ 追加の難関ミッション
- Stage 5: 【ボスミッション】3人全員の相性コンボが必須。クリアで特大ボーナス
