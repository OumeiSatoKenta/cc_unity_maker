# Requirements: Game096_DualControl

## ゲーム概要
2つのキャラクターを左右の親指で同時に独立操作する二刀流アクション。
左半分タップで左キャラ、右半分タップで右キャラを操作。
落下する障害物をそれぞれ避けながら、両キャラが目標数をクリアしたらゴール。

## コアループ
1. 左側タップ → 左キャラが対応する列に移動
2. 右側タップ → 右キャラが対応する列に移動
3. 障害物（赤/紫）が各レーンの列に落下してくる
4. 両キャラとも8個の障害物を回避 → クリア
5. どちらかのキャラが障害物に当たる → ゲームオーバー

## クリア条件
- 左レーン・右レーン それぞれ8個の障害物を回避

## ゲームオーバー条件
- どちらかのキャラが障害物に接触

## 必要な GameObject
- GameManager (DualControlGameManager)
- DualManager (コアメカニクス・グリッド・入力)
- DualControlUI (UI管理)
- Canvas
- EventSystem

## 操作仕様
| 操作 | 入力 | 動作 |
|------|------|------|
| 左キャラ移動 | 画面左半分をクリック | 左キャラが対応列に移動 |
| 右キャラ移動 | 画面右半分をクリック | 右キャラが対応列に移動 |

## グリッド仕様
- 左レーン: 3列 (x = -2.4, -1.6, -0.8)
- 右レーン: 3列 (x = 0.8, 1.6, 2.4)
- キャラクターY位置: -3.5
- 障害物スポーン: y = 6.0
- 障害物消滅: y = -4.8
- 初期速度: 2.5 units/s → 最大5.0 units/s
- 初期スポーン間隔: 1.8s → 最小0.7s

## タイマー
- 開始から計測、クリア時にタイムを表示

## UI 要素
- タイマー（上部中央）
- 左カウンター（左 0/8、上部左）
- 右カウンター（右 0/8、上部右）
- クリアパネル（タイム表示・リトライボタン）
- ゲームオーバーパネル（リトライボタン）
- メニューへ戻るボタン
