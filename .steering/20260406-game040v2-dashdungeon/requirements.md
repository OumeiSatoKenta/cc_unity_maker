# Requirements: Game040v2_DashDungeon

## ゲーム概要

- タイトル: DashDungeon
- カテゴリ: action (v2 remake)
- コアループ: スワイプ（またはタップ）でダッシュ方向を指定し、壁に当たるまで直進しながらダンジョンの出口を目指す

## コアループ

1. プレイヤーが上下左右のいずれかの方向をタップで指定
2. キャラが指定方向に壁まで直進（障害物で止まる）
3. トラップや敵を避けながら出口マスに到達でクリア
4. HPが0になるとゲームオーバー

## クリア条件・ゲームオーバー条件

- **クリア**: 出口マスに到達する
- **ゲームオーバー**: HP が 0 になる

## 必要な GameObject 一覧

- GameManager（DashDungeonGameManager + StageManager + DashDungeonMechanic + DashDungeonUI）
- グリッドタイル（床/壁/トラップ/出口/ワープ/氷床）
- プレイヤーキャラ
- 敵キャラ（敵1種: 通常敵）
- カメラ、Canvas、EventSystem

## 操作仕様

- **タップ（上下左右ボタン）**: 指定方向にダッシュ
  - 上・下・左・右の4方向ボタンをCanvasに配置
  - `Mouse.current.leftButton.wasPressedThisFrame` で入力検知

## チュートリアル表示内容

- **タイトル**: DashDungeon
- **説明**: ダッシュしてダンジョンを攻略しよう
- **操作方法**: 上下左右ボタンで壁まで直進
- **ゴール**: トラップを避けて出口にたどり着こう

## ゲーム深化設計

### 判断ポイント

- **ルート選択**: 最短手数で出口を目指すルート vs トラップを避ける安全ルートの選択。手数が少ないほどボーナス得点が高い
- **HP管理**: 敵やトラップに当たってでも最短ルートを取るか、HPを温存して迂回するか
- **氷床・ワープ活用**: 通常の壁より遠く滑る氷床、ワープ床を逆利用したショートカット

### スキルカーブ

- 下手なプレイヤー: トラップや敵に当たりHPを消耗して詰む
- 上手なプレイヤー: グリッド全体を俯瞰して最短ルートと安全ルートを瞬時に判断。氷床・ワープを逆用してショートカット

### 適用ゲームデザインパターン

**リスク vs リワード選択 + パターン認識 + 応用**
- 最短手数クリアほどボーナスが高いが、トラップ・敵のリスクが高い
- 各ステージで新要素（棘/敵/氷床/ワープ）が追加され、既存の最適解が通用しなくなる

### コンボ/スコアシステム

- 基本スコア: 100pt（クリアボーナス）
- 手数ボーナス: (最短手数との差) × (-5pt)
- HP残存ボーナス: 残りHP × 50pt
- ステージクリアごとに加算

### ステージ別新ルール追加

- Stage 1: 壁ダッシュの基本操作（チュートリアル）
- Stage 2: 棘トラップ追加（踏むとHP-1）
- Stage 3: 敵追加（衝突するとHP-1）
- Stage 4: 氷床追加（着地せず+2マス余分に滑る）
- Stage 5: ワープ床追加（乗ると対になるワープ先に飛ばされる）
