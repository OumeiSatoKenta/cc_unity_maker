# 設計: 全100ゲームのIssue一括登録スクリプト

## 作成ファイル

```
scripts/
├── game-data.tsv          # 100件のゲームデータ（TSV）
└── create-all-issues.sh   # Issue一括登録スクリプト
```

## game-data.tsv のフォーマット

```
ID	タイトル	コアメカニクス	カテゴリ	工数
001	BlockFlow	色付きブロックをスワイプして同色を全て繋げる	puzzle	S
002	MirrorMaze	鏡を配置してレーザーをゴールへ誘導する	puzzle	M
...
```

- ideas.md から手動で抽出（100行）
- タブ区切りで解析しやすい形式

## create-all-issues.sh の設計

### フロー
1. gh認証確認
2. game-data.tsv を1行ずつ読み込み
3. 同タイトルのIssueが既に存在するか確認（冪等性）
4. 存在しなければ Issue を作成（ラベル付き）
5. 1件ごとに1秒スリープ（レート制限対策）
6. 進捗を表示（X/100）
