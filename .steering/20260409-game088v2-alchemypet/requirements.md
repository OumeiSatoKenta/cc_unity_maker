# Requirements: Game088v2_AlchemyPet

## ゲーム概要
錬金術で素材を組み合わせてユニークなペットを生み出す育成ゲーム。素材の組み合わせでペットの種類・能力・外見が決まる。未知の組み合わせを探る実験が楽しい。

## コアループ
1. 素材スロットに素材を選択・投入する
2. 錬金ボタンで合成を実行する（成功→ペット誕生 / 爆発→失敗カウント消費）
3. 誕生したペットに餌やりして育て、新しい素材を獲得する
4. 図鑑を埋めながら次のステージへ進む

## クリア条件
- 現ステージのペット図鑑の目標数を達成する（ペットを目標数以上誕生させる）

## ゲームオーバー条件
- 爆発失敗3回で釜が壊れる → ゲームオーバー

## 必要な GameObject 一覧
- AlchemyPetGameManager（ゲーム状態管理）
- AlchemyManager（コアメカニクス・素材管理・合成ロジック）
- AlchemyPetUI（UI管理）
- StageManager（ステージ進行）
- InstructionPanel（チュートリアル）
- Canvas（UI）
- EventSystem

## 操作仕様
- 素材ボタンをタップ → スロットに素材を選択
- 錬金ボタンタップ → 合成実行
- 餌やりボタン → ペットに餌を与え成長促進

## チュートリアル表示内容
- タイトル: AlchemyPet
- 説明: 錬金術でユニークなペットを生み出そう
- 操作方法: 素材ボタンで選択して投入、錬金ボタンで合成実行
- ゴール: 素材を組み合わせてペット図鑑をコンプリートしよう

## ゲーム深化設計

### 判断ポイント
- 確実なレシピ（既知）に素材を使うか、未知の組み合わせを実験するか（リスクvsリワード）
- 残り失敗回数（3回制）を考慮して攻めの調合か安全策かを判断
- どのペットを優先育成するか（レア素材生産ペットvs図鑑埋め用ペット）

### スキルカーブ
- うまいプレイヤー: 素材の組み合わせパターンを推測し、効率よく新ペットを発見
- 下手なプレイヤー: ランダムに試して爆発ペナルティを重ねる

### ゲームデザインパターン
**リスクvsリワード + パターン認識+応用**
- 未知の素材組み合わせ → 爆発リスクがあるが成功でレアペット入手
- 既知レシピのヒントシステムで少しずつ推測を誘導

### コンボ/乗算スコアシステム
- 連続成功合成でコンボカウント増加
- コンボ×1: 1.0倍、コンボ×2: 1.3倍、コンボ×3: 1.6倍、コンボ×4以上: 2.0倍
- 爆発（失敗）でコンボリセット

### ステージごとの新ルール追加
- Stage 1: 基本2素材合成のみ（チュートリアル的）
- Stage 2: 育成システム追加（ペットに餌やりすると素材ドロップ）
- Stage 3: 3素材合成スロット解禁（より多くの組み合わせ）
- Stage 4: 危険素材登場（失敗確率UP、成功時レア確定）
- Stage 5: 伝説ペット（3連続成功コンボ状態でのみ誕生する特殊ペット）
