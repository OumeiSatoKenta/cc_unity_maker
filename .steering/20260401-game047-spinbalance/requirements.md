# 要求仕様: Game047 SpinBalance

## ゲーム概要
画面をドラッグして盤面を回転させ、端から落ちそうなコマを盤面上にキープし続けるバランスゲーム。コマが増えるほど難易度が上がる。

## コアループ
1. 盤面をドラッグで回転させてバランスを取る
2. コマが端から落ちないよう調整する
3. 制限時間（30秒）まで全コマを保持し続ける
4. 一定間隔で新しいコマが追加される

## クリア条件
- 制限時間（30秒）終了時にコマが1つ以上残っている

## ゲームオーバー条件
- 全てのコマが盤面から落下した

## 操作仕様
| 操作 | 入力 | 動作 |
|------|------|------|
| 盤面回転 | ドラッグ（左右） | 盤面がZ軸で回転 |

## 必要な GameObject
| 名前 | 役割 |
|------|------|
| Board | 回転する盤面（Kinematic Rigidbody2D） |
| Piece | 盤面上のコマ（Dynamic Rigidbody2D） |
| FallZone | 画面外落下検知用トリガー |
| GameManager | ゲーム状態管理 |
| BalanceManager | コアメカニクス（入力・盤面回転・コマ管理） |
| SpinBalanceUI | UI表示 |

## UI要素
| 要素 | 種類 | 初期値 |
|------|------|--------|
| タイマー | Text | 30.0s |
| コマ数 | Text | 残り: N |

## 実装上の注意
- Rigidbody2D でコマの物理挙動
- 盤面は Kinematic で回転させる
- コマは Dynamic Rigidbody2D で重力影響を受ける
- FallZone（画面下部のトリガー）でコマ落下を検知
