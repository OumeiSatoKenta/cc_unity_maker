# Requirements: Game059v2_WaterJug

## ゲーム概要
異なるサイズのジャグ（容器）間で水を移し替え、指定量の水を正確に計量するパズルゲーム。
タップ2回操作で直感的に水を移し替えられる。

## コアループ
1. 目標量とジャグの容量を確認する
2. ジャグをタップして注ぎ元→注ぎ先を選択する（タップ2回）
3. 水が移し替えられる（注ぎ先が満杯か注ぎ元が空になるまで）
4. ターゲットジャグに目標量ぴったりの水が入ったらクリア

## クリア条件
- ターゲットジャグに指定量の水を正確に計量する

## ゲームオーバー条件
- 手数制限を超過（Undoで巻き戻し可能・リトライ無制限）

## 必要なGameObject一覧
- Camera (orthographic, size=6)
- Background
- GameManager (WaterJugGameManager + StageManager + WaterJugUI)
- Jug1, Jug2, Jug3, Jug4（ステージに応じた数）
- FaucetButton（蛇口：タップで注水モード）
- DrainButton（排水口：タップで排水モード）
- Canvas (UI)
- InstructionCanvas (InstructionPanel)
- EventSystem

## 操作仕様
| 操作 | 入力方法 | 操作後の動作 |
|------|---------|-------------|
| 水を注ぐ | タップ2回 | 1回目=注ぎ元選択（ハイライト）、2回目=注ぎ先で水が移動 |
| 蛇口から注水 | 蛇口アイコンタップ→ジャグタップ | ジャグを満杯にする |
| 排水 | ジャグタップ→排水口タップ | ジャグを空にする |
| Undo | Undoボタン | 1手前に戻る |

## チュートリアル表示内容
- **タイトル**: WaterJug
- **説明**: ジャグを傾けて指定量の水を正確に計ろう！
- **操作方法**: ジャグをタップして注ぎ元→注ぎ先を選択。蛇口で満杯、排水口で空にできる
- **ゴール**: 目標ジャグにぴったりの量の水を入れてステージクリア！

## ゲーム深化設計

### 判断ポイント
プレイヤーが毎手ごとに迫られる判断:
- **どのジャグに移すか**: 複数の移し先がある場合、どの組み合わせで目標量を作り出すかの選択
- **いつ排水するか**: ジャグを空にするタイミングが手数効率に大きく影響する（早すぎると手数を無駄にし、遅すぎると行き詰まる）
- **何を犠牲にするか**: 手数を節約するために大胆な注水をするか（リスク高・手数少）、確実に1手ずつ進めるか（リスク低・手数多）
- **Undo戦略**: ミスした時にUndoを使うか（ノーアンドゥボーナス放棄）、リトライするか

### スキルカーブ
- **初心者**: 全手順を確認してから動く、Undoを多用する
- **中級者**: 2〜3手先を読んで動く、最短手数を意識する
- **上級者**: 容量差を利用した最短経路を頭の中で計算できる、ノーアンドゥクリア狙い

### 相互作用するメカニクス: パターン認識+応用
水の移し替えにおける基本パターン（満杯にする→移す→空にする）を認識し、容量差を利用した計量テクニックに応用する。ステージが進むと初期条件や同時目標で応用力が問われる。

### コンボ/乗算スコアシステム
- 連続ステージクリアで累積ボーナス（×1.1〜×1.5）
- 最適解ボーナス（最少手数クリア → ×3.0倍）
- ノーアンドゥボーナス（Undo未使用 → ×1.5倍）

### ステージごとの新ルール追加
- Stage 1: 基本ルールのみ（蛇口・移し替え・排水チュートリアル）
- Stage 2: 手数制限の厳格化（パズルとしての難易度導入）
- Stage 3: 3個目のジャグ追加（3者間の移し替えが必要）
- Stage 4: 初期水量あり（逆算思考が必要）
- Stage 5: 2つの目標量を同時達成（複合チャレンジ）
