# テーマファクトリースキル

このスキルは、プロフェッショナルなフォントとカラーテーマのキュレーションコレクションを提供します。各テーマには慎重に選ばれたカラーパレットとフォントペアリングが含まれています。テーマを選択すると、あらゆるアーティファクトに適用できます。

## 目的

プレゼンテーションのスライドデッキやその他のアーティファクトに一貫したプロフェッショナルなスタイリングを適用する場合にこのスキルを使用する。各テーマには以下が含まれる:
- ヘックスコード付きの統一されたカラーパレット
- ヘッダーとボディテキスト用の補完的なフォントペアリング
- さまざまなコンテキストとオーディエンスに適した独自のビジュアルアイデンティティ

## 使用手順

スライドデッキやその他のアーティファクトにスタイリングを適用する場合:

1. **テーマショーケースを表示する**: `theme-showcase.pdf` ファイルを表示して、ユーザーが利用可能なすべてのテーマを視覚的に確認できるようにする。ファイルに変更を加えず、閲覧のみを目的として表示する。
2. **選択を確認する**: デッキに適用するテーマを確認する
3. **選択を待つ**: 選択されたテーマについて明示的な確認を得る
4. **テーマを適用する**: テーマが選択されたら、選択されたテーマのカラーとフォントをデッキ/アーティファクトに適用する

## 利用可能なテーマ

以下の10種類のテーマが利用可能（`theme-showcase.pdf` にショーケースされています）:

1. **Ocean Depths（オーシャンデプス）** - プロフェッショナルで落ち着きのある海洋テーマ
2. **Sunset Boulevard（サンセットブルバード）** - 暖かく鮮やかなサンセットカラー
3. **Forest Canopy（フォレストキャノピー）** - 自然でグラウンドされたアーストーン
4. **Modern Minimalist（モダンミニマリスト）** - クリーンでコンテンポラリーなグレースケール
5. **Golden Hour（ゴールデンアワー）** - リッチで暖かい秋のパレット
6. **Arctic Frost（アークティックフロスト）** - クールでクリスプな冬にインスパイアされたテーマ
7. **Desert Rose（デザートローズ）** - ソフトで洗練されたダスティトーン
8. **Tech Innovation（テックイノベーション）** - 大胆でモダンなテック美学
9. **Botanical Garden（ボタニカルガーデン）** - フレッシュでオーガニックなガーデンカラー
10. **Midnight Galaxy（ミッドナイトギャラクシー）** - ドラマチックでコスミックなディープトーン

## テーマの詳細

各テーマは `themes/` ディレクトリに完全な仕様で定義されており、以下を含む:
- ヘックスコード付きの統一されたカラーパレット
- ヘッダーとボディテキスト用の補完的なフォントペアリング
- さまざまなコンテキストとオーディエンスに適した独自のビジュアルアイデンティティ

## 適用プロセス

希望するテーマが選択された後:
1. `themes/` ディレクトリから対応するテーマファイルを読み込む
2. 指定されたカラーとフォントをデッキ全体に一貫して適用する
3. 適切なコントラストと可読性を確保する
4. テーマのビジュアルアイデンティティをすべてのスライドで維持する

## カスタムテーマの作成

既存のテーマのいずれもアーティファクトに合わない場合に対応するため、カスタムテーマを作成する。提供された入力に基づいて、上記と同様の新しいテーマを生成する。フォント/カラーの組み合わせが表すものを説明するような類似の名前をテーマに付ける。提供された簡単な説明を使って適切なカラー/フォントを選択する。テーマを生成した後、レビューと検証のために表示する。その後、上記の手順でテーマを適用する。
